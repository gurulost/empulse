This is specifically aimed at the current live symptom:

**Deploy Agent Instructions**
Deploy `origin/main` at commit `f968478`.

Run, in order:

```bash
git fetch origin

composer install --no-dev --prefer-dist --optimize-autoloader
npm ci
npm run build
php artisan migrate --force
php artisan optimize:clear
```

Then restart:
- web process
- queue worker
- scheduler

If your deploy platform supports cache warmup, after `optimize:clear` run:

```bash
php artisan config:cache
php artisan route:cache
php artisan view:cache
```
